### [N1] Note de sensibilité — d'où vient la négativité de masse (clôture cycle 2)

Décomposition sur les 58 282 estimables (prix sortie = médiane DVF existant, coût 2 550 €/m²) :
- **90,2 % des 98,8 % négatives ont la charge foncière DÉJÀ négative** (coût de construction > CA net/m²) —
  seulement **9,8 %** sont négatives à cause du prix du terrain. La marge est donc écrasée **avant** le foncier.
- Prix de sortie **médian = 2 302 €/m²** ; seuil de charge positive = **3 712 €/m²** → le prix de sortie est
  **38 % sous le seuil**. Le coût 2 550 €/m² est une valeur Réunion défendable (fourchette 2300-2800).
- Contrefactuels : coût 2 550→2 000 → 5,9 % positives ; prix sortie ×1,25 → 7,3 % ; ×1,40 → 13,6 %.
- **Verdict : c'est le PRIX DE SORTIE qui écrase**, pas le coût. Il utilise les médianes DVF de l'EXISTANT
  (ancien + maisons, €/m² dilué) là où un promoteur vend du **NEUF/VEFA** (~+30-40 %). Le coût est ~juste ;
  le prix de sortie est **mal spécifié** (mauvais produit).
- **Priorité V2 avant toute exposition client** : calibrer le **prix de sortie par secteur en NEUF/VEFA**
  (pas la médiane tous-biens), avant même la calibration du coût de construction.
